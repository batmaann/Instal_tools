# Instal_tools



### Инструмент для виртуалок

Запуск приложения:
Клонируйте репозиторий с помощью командой в терминале:
```...$  git clone https://github.com/batmaann/Instal_tools.git```
а после перейдите в раздел проекта и в нём выполните команду ```./M1.sh```
